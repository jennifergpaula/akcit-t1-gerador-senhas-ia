# Prompts Utilizados

Este arquivo registra os prompts usados durante o desenvolvimento do projeto.

## Prompt original do usuário

- Criar um gerador de senhas seguras com base em critérios definidos pelo usuário (tamanho, caracteres especiais, números, letras maiúsculas/minúsculas).
- Implementação em Python ou JavaScript, com CLI ou web simples.
- Aplicar IA generativa no desenvolvimento: lógica de geração, testes, interface CLI/HTML.

## Prompt do commit

- Fazer um commit explicando o que foi feito e por que.
- Fazer um novo commit incluindo tudo e explicando o que foi feito.
- Salvar no Git.
- Criar uma pasta e fazer a documentação do projeto.
- Criar uma pasta com os prompts utilizados.

## Observações

A pasta `prompts/` foi criada para manter esses registros organizados e permitir referência futura às instruções de desenvolvimento.
